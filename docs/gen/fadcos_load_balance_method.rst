fadcos_load_balance_method -- configure LB method in Fortinet's FortiADC
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 1.1.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Configure LB method of Application Resources in Server Load Balance Page 



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.8
